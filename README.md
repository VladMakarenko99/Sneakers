# Sneakershop

## Web apllication, build with C#, ASP.NET MVC, Entity Framework Core, PostgreSQL, HTML, CSS, JavaScript. Containerized with Docker and deployed on AWS. It ºs an online store, where the user can filter, search and order sneakers. The user can also authorize in order not to lose the products he previously added to his cart. Additionally, user is able to manage his credentials, upload profile photo, view orders that he made, delete his account.
